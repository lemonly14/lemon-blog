这一节我们主要在node.js里面代码改动后，如何让node server自动重启? 之前我们代码改动后还需要在命令行手动输入 node app.js 重启。

# 断点调试

再讲node server自动重启之前需要把vscode 代码调试讲一下。

代码行的左侧数字旁可以打断点，然后按 F5 调试、也可以使用最左侧的小爬虫调试。

其实呢自动重启是比较简单的，但是要把调试和自动重启结合在一起还是需要做一些代码上的配置的。

# 自动重启

自动重启需要一个nodemon 的 npm 包

在命令行输入npm install nodemon -g 全局安装

提示

使用 -g 安装的 npm 包是不会出现在package.json里，因为它是全局安装的不然只是在本项目中使用的

那我们如何使用 nodemon 呢？

我们直接在命令行输入 nodemon app.js

成功！

我们测试一下 我们修改了代码整个 node server 可以自动重启吗？ 我们在某一行加一个空格 然后 ctrl+s 看一下效果

那么我既想进行断点调试又想整个 node server 可以重启呢？

可以的。我们在vscode里可以通过配置文件来决定一个项目的启动方式。

那么如何为vscode添加一个启动配置文件呢?

最简单的方式是点击这里的小爬虫再点击绿色按钮里面的添加配置

点击完成后 vscode 会在项目里添加一个**.vscode 文件夹** 里面有launch.json文件。可以在其中配置我们当前项目的启动方式。
