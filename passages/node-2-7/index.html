# 强制 Promise

我们知道第一个中间件通过 next()调用了第二个中间件，那么这个 next()有返回结果吗？如果有，那么返回的结果是什么呢？

我们来验证一下：

app.use((ctx, next) => {
  console.log(1)
  const a = next()
  console.log(a)
  console.log(2)
})

app.use((ctx, next) => {
  console.log(3)
  // next()
  console.log(4)
})

我们更改一下代码之后重启一下服务，再使用浏览器发送一下http请求，我们可以看到输出了一个Promise{undefined}

我们可以看到a 打印出来的是一个Promise，返回的是一个undefined，这是因为第二个中间并未返回什么东西。

那我return一些东西看看效果...

app.use((ctx, next) => {
  console.log(1)
  const a = next()
  console.log(a)
  console.log(2)
})

app.use((ctx, next) => {
  console.log(3)
  // next()
  console.log(4)
